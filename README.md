# Telegram Crypto Bot (C++) 

Простой Telegram-бот на C++, который демонстрирует работу с современным шифрованием. Бот принимает текстовые сообщения, шифрует их по алгоритму **AES-256 (GCM)** и умеет расшифровывать их обратно.

##  Возможности

*   **Шифрование:** Использует библиотеку Botan для реализации AES-256-GCM.
*   **Безопасность:** Для каждого сообщения генерируется уникальный `nonce` (IV).
*   **Интеграция:** Работает через Telegram Bot API (Long Polling).
*   **Формат:** Передает зашифрованные данные в кодировке Base64.

##  Стек технологий

*   **Язык:** C++17
*   **Криптография:** [Botan](https://randombit.net)
*   **Сеть:** [libcurl](https://curl.se)
*   **JSON:** [nlohmann/json](https://github.com)

##  Предварительные требования

Для сборки вам понадобятся установленные библиотеки.

**В Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install libbotan-2-dev libcurl4-openssl-dev nlohmann-json3-dev build-essential
```

### Настройка и запуск

1. **Получите токен бота**: Создайте нового бота через [@BotFather](https://t.me).
2. **Установите токен**: Замените `YOUR_BOT_TOKEN_HERE` в файле `src/main.cpp` на ваш реальный токен.
3. **(Опционально) Смените ключ**: Замените значение `keyHex` в коде (сейчас используется пример `0123456789ABCDEF...`). Ключ должен быть длиной 32 байта (64 символа в Hex).

**Запуск:**
```bash
./build/bot
```
**Пример успешного запуска:**
```bash
Бот запущен. Используется Botan 2.19.3
```
# Использование в Telegram
1. Начните диалог с ботом
Отправьте боту команду /start

2. Шифрование сообщения
Отправьте любой текст → бот зашифрует его AES-256-GCM и вернет Base64 строку

3. Расшифровка сообщения
Отправьте /decrypt <base64_строка> → бот расшифрует сообщение

Пример диалога
```bash
text
Пользователь: /start
Бот: Пришлите текст для шифрования или /decrypt <base64> для расшифровки

Пользователь: Привет, мир!
Бот:  Зашифровано (AES-256-GCM):
`Gk8sLd9FjK2mNpQrStUvWxYz1234567890ABCDEF=`
Отправьте /decrypt Gk8sLd9FjK2mNpQrStUvWxYz1234567890ABCDEF= для расшифровки

Пользователь: /decrypt Gk8sLd9FjK2mNpQrStUvWxYz1234567890ABCDEF=
Бот:  Расшифровано: Привет, мир!
```
